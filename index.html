My own elevator

A simple website navigation inspired by Kirkas
It supports key navigation, and works smoothly on handheld devices such as iPads or Iphones (swipe left/right to change slide or swipe up/down to change project)
It's supposed to work as an image gallery for portfolios so it has a neat preloader to let the user know how much he has left to wait.
Other features include direct linking to a specific slide, full responsiveness, dynamic choco-navigation menu creation(look at the bottom).
It uses css3 transitions if the browsers supports them, otherwise it will fall back to jQuery animation, this means it will work ie7+, chrome, firefox and any other browser.
